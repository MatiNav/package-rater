# Nest Starter Template

1. Clone Project
2. Install dependencies
```
npm i
```
3. Clone `.env.template` and rename it to `.env`
4. Put env variables in `.env`
5. Start the database (with syncrhonization): 
```
docker-compose up -d
```
6. Run app in dev mode
```
npm run start:dev
```
